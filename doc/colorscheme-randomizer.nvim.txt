1. colorscheme-randomizer.nvim*colorscheme-randomizer.nvim-colorscheme-randomizer.nvim*

`colorscheme-randomizer.nvim` introduces the ability to randomly apply various
available colorschemes.


FEATURES    *colorscheme-randomizer.nvim-colorscheme-randomizer.nvim-features*

- Randomly apply available colorschemes
- Automatically finds colorscheme plugins
    - `lazy.nvim` supported
- API to apply more colorschemes randomly


REQUIREMENTS*colorscheme-randomizer.nvim-colorscheme-randomizer.nvim-requirements*

- Modern Neovim


OPTIONAL    *colorscheme-randomizer.nvim-colorscheme-randomizer.nvim-optional*

- `lazy.nvim` package manager to automatically find installed colorschemes
    - More package manager support to be added later


INSTALLATION*colorscheme-randomizer.nvim-colorscheme-randomizer.nvim-installation*

    require("colorscheme-randomizer").setup(opts)

2. Configuration                   *colorscheme-randomizer.nvim-configuration*

  -------------------------------------------------------------------------------------
  Property               Type             Description
  ---------------------- ---------------- ---------------------------------------------
  apply_scheme           boolean?         Apply the colorscheme after being picked.

  colorschemes           table?           list of colorschemes to randomly select from.
                                          When specified, this list is the only source
                                          of truth to choose from.

  exclude_colorschemes   table?           list of colorschemes to remove from
                                          colorschemes. useful when wanting to use
                                          plugin_strategy and ignore some colorschemes.

  plugin_strategy        string?          lazy is the only supported option. Strategy
                                          used to resolve plugin names to colorscheme
                                          names

  plugins                table?           list of colorscheme plugins to look through
                                          to resolve colorschemes. When unspecified,
                                          all colorschemes are attempted to be resolved
                                          via plugin_strategy
  -------------------------------------------------------------------------------------
